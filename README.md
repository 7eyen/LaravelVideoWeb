> LaravelVideo配套前端项目

## 安装

这个项目使用 [node](http://nodejs.org) 和 [npm](https://npmjs.com). 请确保你本地安装了它们。

> 编译前修改`src/mian.js`文件中的`axios.defaults.baseURL`对应域名

```bash
# install dependencies
$ npm install

# build for production with minification
npm run build
```
## 使用说明

编译完成后,将`/disk`文件部署到网站根目录
